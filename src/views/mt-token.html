MT Tokens

Generate codes to receive 50 tokens at once, with automatic weekly rewards of 100 tokens. Use them seamlessly across all platforms and store up to 200 tokens for uninterrupted automation.

What is MT Token

MT Token is a digital access system for MT Auto Clicker services. Each token provides 10 minutes of service usage across Windows applications, Mac applications, and browser extensions. This token system helps manage fair usage and service access for all users. Tokens are uniquely tied to your email address and can be stored up to a maximum of 200 tokens per account.

Important Note: Every user automatically receives 100 tokens each week, whether they use the service or not. These tokens are added to your account balance automatically up to the maximum limit of 200 tokens.

How to Generate Tokens?

Visit the MT Token page

Enter your email address

Click 'Generate Token Code'

You'll receive a unique code valid for 10 minutes

Copy this code

Open your MT Auto Clicker software or extension

Go to the Token page in the software/extension

Paste the code

50 tokens will be added to your account

Remember

  • Codes are valid for 10 minutes only
  • Each code adds 50 tokens to your account.
  • You can generate tokens anytime (up to 200 total tokens).
  • Codes must be used with the same email address used to generate them.

How to Use MT Tokens?

Open your MT Auto Clicker application or extension.

Sign in with your registered email.

Your tokens will automatically sync across all platforms (Windows, Mac, browser extensions) when using the same email.

Each token provides 10 minutes of service access.

Tokens are automatically deducted as you use the service.

Monitor your token balance in the application.

    • Frequently Asked Questions

    What happens if my code expires before I use it?

    If the 10-minute period expires, you'll need to generate a new code.

    Do I need to redeem the code on all my devices?

    No, redeeming the code on any one platform (Windows, Mac, or browser extension) will automatically sync the tokens across all platforms using the same email.

    What if I enter the wrong email when generating a code?

    The code will only work with the email address used to generate it. Make sure to use the same email across all platforms.

    Can I generate multiple codes at once?

    You can generate new codes as long as your total tokens don't exceed 200.

    What happens if I reach the 200 token limit?

    You won't be able to receive new weekly tokens or generate additional codes until you use some of your existing tokens.

    Can I get a refund for unused tokens?

    Tokens are non-refundable, but you can continue using them until depleted.

    What happens if I lose internet connection while using tokens?

    Token validation requires an internet connection. Service access may be interrupted until connection is restored.

    Can I use multiple email addresses to generate more tokens?

    No, creating multiple accounts to circumvent token limitations is prohibited and may result in account restrictions.

    What should I do if my tokens aren't syncing across devices?

    Ensure you're using the same email address on all devices and have an active internet connection. If issues persist, contact support.

    Can I purchase additional tokens?

    Currently, tokens can only be generated through the standard system up to 200 tokens. For unlimited usage, consider upgrading to a Premium subscription.
